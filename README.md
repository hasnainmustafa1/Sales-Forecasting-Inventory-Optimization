# Sales Forecasting and Inventory Optimization (Synthetic Demo)

This project is a demo pipeline for sales forecasting and inventory optimization using a synthetic dataset.

## Quickstart

1. Create venv: python -m venv venv
2. Activate: source venv/bin/activate (Windows: venv\\Scripts\\activate)
3. Install: pip install -r requirements.txt
4. Train model: python src/train_forecast_model.py
5. Evaluate: python src/evaluate_model.py
6. Inventory report: python src/inventory_optimizer.py
7. Run API: python app.py
